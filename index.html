native2ascii and vice versa

This is a pure JavaScript implmentation of native2ascii, convert character to UTF8-Decoded string, and vice versa.

install and usages

Install it from npm.

$ npm install native2ascii

It simply provided two functions for converting.

assert(require('native2ascii').native2ascii('中'), '\\u4e2d')
assert(require('native2ascii').ascii2native('\\u4e2d'), '中')
